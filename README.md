-- QuickMart Inventory Management System --

## Overview
QuickMart Inventory Management System is a relational database application designed for a small grocery store. The system helps manage products, inventory, suppliers, customer orders, deliveries, and reorder requests.

## Problem Statement
Small retail stores often rely on manual tracking or spreadsheets for inventory management. This can cause stock shortages, poor reorder tracking, and inefficient reporting. QuickMart solves this by centralizing business data in a relational database.

## Features
- Track products, categories, suppliers, and inventory levels
- Manage customer orders and order items
- Record deliveries
- Generate reorder requests for low-stock products
- Run business reports for inventory and sales analysis

## Technologies & Diagrams Used
- Microsoft Access
- SQL
- Relational Database Design
- ER Modeling
- Normalization

## Database Tables
- Customers
- OnlineAccounts
- Orders
- OrderItems
- Products
- Categories
- Suppliers
- Inventory
- Delivery
- ReorderRequests

## Example Business Queries
- Low-stock inventory report
- Pending reorder requests
- Customer order summaries
- Product sales by category
- Supplier inventory report
